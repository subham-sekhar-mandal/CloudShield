# CloudShield: Cloudburst Early Warning System

# Overview

CloudShield is a frontend-based web application that simulates a Cloudburst Early Warning and Alarm System.
It monitors environmental parameters like rainfall, humidity, and wind speed, and predicts the risk level of a cloudburst using a rule-based logic system.

This project is built using HTML, CSS, JavaScript, and React, without relying on external APIs or libraries.

# Objective

The goal of this project is to:

Demonstrate how early warning systems for natural disasters can be designed
Simulate real-time environmental monitoring
Provide alerts for potential cloudburst conditions
Build an interactive and responsive dashboard using core frontend technologies

# Tech Stack
HTML5 – Structure
CSS3 – Styling and UI
JavaScript (ES6) – Logic and simulation
React.js

# Features
Simulated Weather Data
Generates dynamic weather data (rainfall, humidity, wind speed)
Updates automatically at regular intervals

# Risk Detection System
Calculates cloudburst risk using predefined thresholds:
🟢 Low Risk
🟡 Medium Risk
🔴 High Risk

# Alert & Alarm System

Displays warning popups for high-risk situations
Plays alarm sound for emergency alerts
Visual indicators (color-coded warnings)
